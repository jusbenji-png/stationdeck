Select Excel File
What happens when you upload?

1. The file is saved to data/input/
2. All daily records are read and imported into the database
3. Existing records for the same dates are updated (no duplicates)
4. You can then generate reports for any month in the file
Accepted File Format
The file must be the Daily Cash Flow Excel workbook with a sheet named CASH FLOW.

It should contain columns including: date, PMS volume, AGO volume, revenues, expenses, cash banked, and delta.

Files with multiple months of data are fully supported — all records will be imported at once.
